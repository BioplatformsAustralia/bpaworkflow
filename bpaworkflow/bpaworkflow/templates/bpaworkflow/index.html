Upload verification system

Sequencing facilities providing data to BioPlatforms Australia can use this page to confirm that metadata requirements, and file naming conventions, have been followed. Any issues identified can then be addressed, prior to transfer of the data to Bioplatforms.

Select the Project for which data has been generated, and then the Import type for the data being uploaded. This will be described by the -omics (e.g. genomics, transcriptomics, ...) and the sequencing technology in use.

Select your submission sheet (an Excel file, XLSX format), and the MD5 checksum file (a text file of MD5 checksums, in Linux or BSD/MacOS format), then press verify to validate your upload.
